行政書士の勉強スケジュール！1年間の学習計画を月別に解説

行政書士試験に合格するには、計画的な学習スケジュールが不可欠です。試験日は毎年11月の第2日曜日で、1年前から逆算して月別の計画を立てることが最も効率的です。この記事では1年間の月別学習スケジュール、科目の学習順序、3フェーズ構成の学習法を解説します。

行政書士試験の年間スケジュール

まず、2026年度の試験スケジュールを確認しましょう。

| イベント | 時期 | |---------|------| | 試験公告 | 2026年7月上旬（予定） | | 申込期間 | 2026年7月下旬〜8月下旬（予定） | | 試験日 | 2026年11月8日（日）予定 | | 合格発表 | 2027年1月下旬（予定） |

行政書士の試験日程2026で最新の日程を確認してください。

学習開始時期は試験の12ヶ月前（前年の11月〜12月）が理想的です。社会人が1日2〜3時間の学習で800時間を確保するには、12ヶ月の期間が必要です。

3フェーズ構成の学習法

1年間の学習を3つのフェーズに分けて進めます。

| フェーズ | 期間 | 内容 | 時間配分 | |---------|------|------|---------| | Phase 1：基礎インプット | 12〜7ヶ月前 | テキスト通読・基本理解 | 全体の40% | | Phase 2：過去問演習 | 7〜3ヶ月前 | 問題演習・知識定着 | 全体の35% | | Phase 3：直前対策 | 3ヶ月前〜試験日 | 模試・弱点補強・記述式 | 全体の25% |

このフェーズ構成のメリットは、各段階で明確な目標を設定できることです。Phase 1で「理解」、Phase 2で「定着」、Phase 3で「実戦力」を順番に積み上げていきます。

Phase 1：基礎インプット期（12〜7ヶ月前）

目標：テキスト全体を2周し、各科目の全体像を掴む

月別の学習内容：

1ヶ月目（12ヶ月前）：民法（総則・物権） 民法から始める理由は、行政法の理解に民法の基礎知識が必要だからです。まず総則（権利能力・意思表示・代理）と物権（所有権・抵当権）を学びます。

2ヶ月目（11ヶ月前）：民法（債権・親族相続） 債権（契約・不法行為・債務不履行）と親族相続を学びます。民法は行政書士の勉強法でも最重要科目として位置づけています。

3ヶ月目（10ヶ月前）：行政法（行政法総論・行政手続法） 行政法に入ります。まず行政法総論で行政行為の概念を理解し、行政手続法の条文を押さえます。

4ヶ月目（9ヶ月前）：行政法（行政事件訴訟法・行政不服審査法） 行政法の中核となる訴訟法と不服審査法を集中的に学びます。条文の正確な理解が記述式にも直結します。

5ヶ月目（8ヶ月前）：行政法（国家賠償法・地方自治法）＋憲法 行政法の残りの分野と憲法を並行して学びます。憲法は人権分野の判例が中心です。

6ヶ月目（7ヶ月前）：商法・基礎法学＋テキスト2周目開始 商法は基本的な条文と会社法の基礎を押さえます。同時にテキスト2周目に入り、理解度を確認します。

Phase 2：過去問演習期（7〜3ヶ月前）

目標：過去問5年分を3周し、知識を実戦レベルまで引き上げる

月別の学習内容：

7ヶ月目（6ヶ月前）：過去問1周目（行政法・民法） 行政書士の過去問の使い方を参考に、まず行政法と民法の過去問を解きます。1周目は正答率を気にせず、出題パターンを把握することが目的です。

8ヶ月目（5ヶ月前）：過去問1周目（憲法・商法・一般知識）＋2周目開始 残りの科目の過去問を解き、すぐに2周目に入ります。1周目で間違えた問題に印をつけておき、2周目では重点的に復習します。

9ヶ月目（4ヶ月前）：過去問2〜3周目＋一般知識対策開始 過去問の2周目を完了し、3周目に入ります。同時に一般知識の対策を本格的に開始します。行政書士の足切りを回避するために、文章理解の練習を毎日1問ずつ始めましょう。

10ヶ月目（3ヶ月前）：過去問3周目完了＋記述式対策開始 過去問3周目を完了します。この段階で正答率70%以上を目指します。記述式の対策もこの時期から本格化させます。

Phase 3：直前対策期（3ヶ月前〜試験日）

目標：模試で180点以上を安定して取れる状態にする

月別の学習内容：

11ヶ月目（2ヶ月前）：模試受験＋弱点科目の集中補強 予備校の模試を最低2回受けましょう。模試で弱点が見つかれば、その科目に集中的に時間を投入します。

12ヶ月目（1ヶ月前）：総復習＋条文暗記＋文章理解仕上げ テキストの重要ポイントを総復習し、記述式で使う条文のキーワードを暗記します。一般知識の文章理解も最終確認を行います。

直前1週間： 新しい論点には手を出さず、これまでの総復習に徹します。過去問で間違えた問題を見直し、体調管理を最優先にしましょう。

社会人向けの時間確保テクニック

フルタイムで働きながら学習時間を確保するための具体的なテクニックです。

平日の学習パターン（目標：2〜3時間）：

  • 朝活：出勤前に1時間（5:30〜6:30）
  • 通勤時間：往復で30分〜1時間（音声講義・一問一答）
  • 夜：帰宅後に1時間（21:00〜22:00）

休日の学習パターン（目標：5〜6時間）：

  • 午前：3時間（9:00〜12:00）
  • 午後：2〜3時間（14:00〜17:00）

行政書士の勉強時間の目安は600〜1,000時間です。上記のペースで12ヶ月継続すれば、約800〜900時間を確保できます。

まとめ

行政書士試験の学習は、3フェーズ構成（基礎インプット→過去問演習→直前対策）で1年間の計画を立てるのが王道です。民法→行政法→憲法→商法の順で学び、試験の6ヶ月前から過去問演習に移行します。計画を立てたら、行政書士通信講座おすすめ比較で自分に合った教材を選び、今日から学習を始めましょう。
